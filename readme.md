# Выполнение Л.Р. №1
---
## Назначение
Программа предназначена для решения лабораторной
работы №1 *"Освоение языка Python"*. по МУАМ.

## Установка
Скачать файл. Выполнить Run_Lab_1.py с помощью Python
> Примечание: Л.Р. выполнено в версии Python 3.8

## Зависимости
Список использованных библиотек:
- NumPy
- MatPlotLib.PyPlot
- Random
- WiZeO_lib (персональная авторская библиотека)

## Структура
**- src:** 9 упражнений ex*.py и библиотека WiZeO_lib
Run_Lab_1.py: собственно, само упражнение.

**- Run_Lab_1.py:** Собственно, который запускает
выполнение лабораторной работы.
